# 1. 프로그램 설계서 (5점)

## 1. 프로젝트 주제
컴퓨터와의 행맨게임(랜덤으로 단어를 뽑아 글자 단위로 맞추는 게임)

## 2. 핵심 기능 목록
(구현할 기능들을 순서대로 나열하세요.)
1.  Strchr 함수를 이용하여 맞춘 글자인지 확인
2.  단어에 포함된 글자일 경우 정답 처리
3.  목숨을 차감하며, 단어를 완성하면 축하 메시지
4.  남은 목숨을 출력하고, 게임이 끝나면 결과를 보여줌
