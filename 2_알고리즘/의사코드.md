# 2. 알고리즘 (의사 코드) (10점)

## 1. `main` 함수의 전체 흐름
시작
    1. 단어 목록에서 랜덤으로 단어를 선택한다.
    2. 남은 목숨을 6으로 초기화한다.
    3. 사용자에게 추측할 글자를 입력받을 공간을 마련한다.
    4. 추측한 글자가 정답인지 확인한다.
    5. 만약 정답이면:
        1. 단어에 포함된 모든 맞춘 글자 표시
        2. 게임 계속 진행
    6. 만약 틀리면:
        1. 목숨 차감
        2. 틀린 글자 출력
        3. 남은 목숨 출력
    7. 만약 사용자가 단어를 다 맞추면:
        1. 축하 메시지 출력
        2. 게임 종료
    8. 만약 목숨이 모두 소진되면:
        1. "게임 오버" 메시지 출력
        2. 게임 종료
반복
종료

## 2. 핵심 기능 1: (예: 학생 등록 함수)
시작
    1. 사용자로부터 추측할 글자 하나 입력받기
    2. `Strchr` 함수를 사용하여, 입력한 글자가 단어에 포함되어 있는지 확인한다.
    3. 만약 글자가 포함되어 있으면:
        1. 맞춘 글자로 간주하여 `guessed` 배열에 기록
        2. 맞춘 글자들을 표시해준다 (단어를 부분적으로 보여줌)
        3. 사용자가 맞춘 글자들을 계속해서 표시
    4. 만약 글자가 포함되어 있지 않으면:
        1. 목숨 하나 차감
        2. 틀린 글자 출력
        3. 남은 목숨 수를 출력한다.
종료

## 3. 핵심 기능 2: (예: 학생 검색 함수)
시작
    1. 단어를 다 맞췄는지 확인한다.
    2. 만약 단어를 모두 맞췄다면:
        1. "축하합니다! 단어를 맞췄습니다!" 메시지 출력
        2. 게임 종료
    3. 만약 목숨이 0이 되면:
        1. "게임 오버! 다시 시도해주세요!" 메시지 출력
        2. 게임 종료
종료
